# 📊 Unemployment Analysis Dashboard

An interactive Data Analytics and Visualization project built using Python, Streamlit, Pandas, and Plotly to analyze unemployment trends across different regions of India.

This dashboard provides detailed insights into unemployment rates, labour participation, state-wise analysis, urban vs rural comparison, and the impact of COVID-19 on unemployment trends using interactive visualizations and data exploration techniques.

---

# 🚀 Features

- 📈 Interactive unemployment trend analysis
- 🏙️ State-wise unemployment comparison
- 🌆 Urban vs Rural analysis
- 🦠 COVID-19 impact visualization
- 👥 Employment vs unemployment analysis
- 📊 Interactive charts using Plotly
- 🔍 Dynamic filtering by region and area
- 🏆 Ranking of states based on unemployment rate
- 📁 Dataset preview and downloadable filtered data
- 🌙 Light and Dark theme support

---

# 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Plotly

---

# 📂 Dataset Information

The dataset contains unemployment-related information such as:

- Region/State
- Date
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)
- Area (Urban/Rural)

---

# 📈 Dashboard Sections

## 📊 Overview
Shows unemployment trends over time and employment analysis.

## 🏙️ State Analysis
Compares unemployment rates across different states and regions.

## 🦠 COVID-19 Impact
Analyzes the effect of COVID-19 on unemployment during 2020.

## 🏆 Ranking Analysis
Displays top states with highest unemployment rates.

## 📁 Dataset Section
Provides dataset preview and download option.

---

# 💡 Key Insights

- COVID-19 significantly increased unemployment rates during 2020.
- Some states consistently showed higher unemployment rates.
- Urban and rural regions displayed different unemployment patterns.
- Labour participation and employment levels directly influenced unemployment trends.
